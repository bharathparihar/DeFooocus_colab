
# B Catalog - Cart Chat Now

## Project info
This is **B Catalog**, a mobile-first catalog shop application for businesses to showcase products/services and receive inquiries via WhatsApp.

**Key Features:**
- **Seller Dashboard**: Manage products, services, gallery, and business info.
- **Buyer Catalog**: Public-facing shop page for customers.
- **PDF Catalog**: Generate professional product lists instantly.
- **Social Embeds**: Link Twitter, Maps, Instagram, Youtube, etc.
- **Analytics**: Track views and clicks.


## Getting Started

To run this project locally, you need Node.js & npm installed.

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd cart-chat-now

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the application for production:

```sh
npm run build
```
